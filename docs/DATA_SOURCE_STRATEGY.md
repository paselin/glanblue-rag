## データ統合戦略

### 同一キャラクターの情報統合

```python
# 例: カタリナ（SSR・水属性）
{
    "name": "カタリナ",
    "sources": {
        "gamewith": {
            "rating": 9.5,
            "tags": ["サポート", "初心者向け"],
            "content": "GameWithの詳細..."
        },
        "wiki": {
            "aliases": ["水カタ", "SSRカタ"],
            "status": "詳細なステータス",
            "content": "Wikiの詳細..."
        }
    },
    # 統合コンテンツ
    "unified_content": "GameWith + Wiki の統合情報"
}
```

### RAG検索での活用

```python
# ユーザー: "水ゾの使い方は？"
# → Wikiから「水ゾ = 水着ゾーイ」を解決
# → GameWithから最新評価を取得
# → 統合して回答
```
